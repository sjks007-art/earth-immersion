## 🎯 왜 3줄 몰입인가?

### 과학적 근거
- 🧠 **1분 = 최소 유효 시간** - 뇌과학이 입증한 최소 집중 단위
- 🌊 **3단계 = 완벽한 리듬** - 황농문 교수 몰입 이론 기반
- 💚 **매일 = 습관의 시작** - 21일 법칙으로 검증된 변화

### 지구에 미치는 영향
```
1명이 1분 실천 → 일회용품 1개 감소
100명이 1분 실천 → 탄소 100kg 감소
10,000명이 1분 실천 → 숲 1개 조성 효과
1,000,000명이 1분 실천 → 지구 온도 0.001°C 하락
```
